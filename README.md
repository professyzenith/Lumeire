## ✦ &nbsp;What Is This

**Lumière** is a cinematic, fully animated one-page restaurant website I built **just for fun** as a personal project and portfolio piece. No client, no brief — just me experimenting with animations, particles, and motion design to see how far I could push pure HTML/CSS/JS.

It's designed to feel like an Awwwards-level site — not because I'm trying to win awards, but because it was genuinely fun to build something that makes people ask *"how was this made?"* 😄

## ✦ &nbsp;The Intro Sequence

<table>
<tr>
<td width="50%">

**Stage 1 — Preloader** `~2s`
- Dark background
- Gold progress line fills 0 → 100%
- Shimmering light tip rides the edge
- Corner brackets snap in
- 12-blade curtain wipe exit

</td>
<td width="50%">

**Stage 2 — Main Intro** `~4.2s`
- 280 ember particles + wave layers
- Shooting stars arc across the void
- Gold sweep beam reveals each letter
- Particle sparks burst per character
- 18-panel film-rip into homepage

</td>
</tr>
</table>

## ✦ &nbsp;Features

<table>
<tr><td>🎬</td><td><strong>Cinematic Two-Stage Intro</strong> — Preloader + LUMIÈRE letter reveal via a sweeping gold beam with particle sparks and film-rip exit</td></tr>
<tr><td>✦</td><td><strong>GSAP ScrollTrigger</strong> — Staggered entrances, clip-path text reveals, parallax scrub, count-up statistics across every section</td></tr>
<tr><td>🖱️</td><td><strong>Custom Cursor</strong> — Dual-layer dot + trailing ring with magnetic elastic snap-back on all primary buttons</td></tr>
<tr><td>🌟</td><td><strong>Canvas Particle Systems</strong> — Two fully hand-coded Canvas 2D fields (hero + footer), zero external particle libraries</td></tr>
<tr><td>🖼️</td><td><strong>Mouse-Reactive Hero</strong> — Radial spotlight tracks cursor, ghost-text parallaxes with mouse movement</td></tr>
<tr><td>💡</td><td><strong>Menu Card Spotlight</strong> — Radial glow follows the cursor independently per card</td></tr>
<tr><td>📱</td><td><strong>Fully Responsive</strong> — Mobile breakpoints in a dedicated <code>responsive.css</code></td></tr>
<tr><td>⚡</td><td><strong>Zero Build Step</strong> — Pure HTML/CSS/JS, no npm, no bundler, just open and run</td></tr>
</table>

## ✦ &nbsp;Project Structure

```
lumiere/
├── index.html                ← Semantic HTML markup
├── css/
│   ├── style.css              ← Core styles — layout, colour, typography, keyframes
│   └── responsive.css          ← Mobile/tablet breakpoints
├── js/
│   ├── utils.js                ← Global helpers + GSAP plugin registration
│   ├── cursor.js                ← Custom cursor — self-contained
│   ├── animations.js             ← Preloader · intro · all ScrollTrigger logic
│   └── main.js                    ← Canvases · magnetic buttons · boot sequence
├── assets/
│   ├── images/                 ← Drop real photos here
│   ├── icons/                  ← Favicon slot
│   └── fonts/                  ← Self-hosted font slot (optional)
└── README.md
```

## ✦ &nbsp;Tech Stack

<div align="center">

| Layer | Technology |
|:---|:---|
| Markup | Semantic HTML5 |
| Styling | Plain CSS3 — no preprocessor |
| Animation | [GSAP 3](https://gsap.com/) + ScrollTrigger via CDN |
| Particles | Hand-written Canvas 2D |
| Typography | Cormorant Garamond + Montserrat (Google Fonts) |
| Build | **None** — open `index.html`, it just works |

```bash
# Node
npx serve .
```

Then visit → `http://localhost:8000`

Or just double-click `index.html` — works fine directly.
