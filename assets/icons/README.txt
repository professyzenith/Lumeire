Reserved for favicon / app icons.

Recommended files to add:
  favicon.ico        — 32x32, referenced automatically by browsers
  favicon.svg        — modern vector favicon (optional)
  apple-touch-icon.png — 180x180, for iOS home screen bookmarks

Once added, link them in index.html <head>:
  <link rel="icon" href="assets/icons/favicon.ico">
  <link rel="apple-touch-icon" href="assets/icons/apple-touch-icon.png">
