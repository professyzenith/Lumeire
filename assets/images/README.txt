This folder is ready to receive your real photography.

Drop in JPG/WebP files using these exact filenames, and they will
automatically appear once you add the "photo-mode" class to <body>
in index.html (see the main README.md → "Enabling Real Photography"):

  about-interior.jpg          — restaurant interior for the Story section
  gallery-grand-hall.jpg      — main dining hall
  gallery-garden-terrace.jpg  — outdoor seating area
  gallery-private-dining.jpg  — private dining room
  gallery-wine-cellar.jpg     — wine cellar / storage

Recommended specs:
  - Minimum 1600px on the longest edge
  - JPG quality 80-85% (or WebP for smaller file size)
  - Landscape orientation works best for all 5 slots

Good free sources for placeholder/demo photography until you have
real shoot photos from the client:
  - unsplash.com   (search: "fine dining interior", "restaurant
                     wine cellar", "private dining room")
  - pexels.com     (same search terms)

Both are free to use commercially with no attribution required.
